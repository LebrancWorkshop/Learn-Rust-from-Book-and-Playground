# Rust: Condition

## If-Else

```rs
if <condition> {
  <expression>
} else if <condition> {
  <expression>
} else {
  <expression>
}
```

## If-Else as Value

```rs
let <variable_name> = if <condition> {
  <expression_as_a_value>
} else if <condition> {
  <expression_as_a_value>
} else {
  <expression_as_a_value>
}
```
